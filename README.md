# Roullete Revolver Game

## Description 
This is a simple web-based game built with React. It is based on how the **Russian Revolver** game works -> It is a game of chance where a player progresses through a series of skill-based and logic-based challenges to reach through the final 'cylinder' stage, where survival is determined by probability.

## How to Install
1. Fork the repository
2. Clone the repository to your local machine :

```bash
git clone git@github.com:cedrickmeka/Roulette-revolver-game-project.git
```
3. Navigate into the project directory and install dependencies :

```bash
npm install
```

## How to run : 

To run the project use the command below and open the localhost provided on the website
```bash
npm run dev
```

## How to play : 
1. Read instructions to understand the game play.
2. Each of two players is to then select their avatar then proceed to the game with the button provided.
3. In the gameplay screen, each player will have a chance to click a button which spins the cylinder and shoots the opponent based on probability.
4. Before the winner is decided, if one player gets the chance to proceed to the next level, he/she has to answer several trivia questions for them to be considered as the winner.
5. The game ends with a winner when the player answers the trivia questions correctly or with a lost game when the trivia answers weren't correct.

## Technologies Used
1. **HTML**  - Page structure
2. **CSS** & **Tailwind CSS**- Styling
3. **React** - Allows for Dynamic Inputs and Updates
